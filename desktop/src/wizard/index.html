Đang chuẩn bị OpenClaw cho lần đầu sử dụng

OpenClaw cần WSL2 + Ubuntu để chạy backend. Quá trình này diễn ra tự động — chỉ mất vài phút và chỉ làm 1 lần.

  1. Kiểm tra hệ thống
  2. Cài WSL2 (cần restart sau)
  3. Tải Ubuntu 22.04
  4. Cấu hình distro
  5. Cài OpenClaw backend
  6. Kiểm tra hoạt động
